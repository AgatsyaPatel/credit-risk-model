# Credit Risk Prediction Model

## Overview
Machine learning model to predict loan defaults 
using real LendingClub data (2.2 million loans).

## Dataset
- Source: LendingClub via Kaggle
- Target: default (1 = Charged Off, 0 = Fully Paid)

## Tools
- Python, Pandas, NumPy
- Scikit-learn, XGBoost
- Matplotlib, Seaborn

## Run Order
01_data_loading → 02_cleaning → 03_features → 04_model
